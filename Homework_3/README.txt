OOP

Software Requirements:

Create a small application that mimics a library catalog. It needs to be able to add books, delete books and list books. Books are of two kinds: novels and art albums. They both have names and number of pages. Novels have type (like mystery, sf, etc.) while albums have paper quality.

Model these entities (book, novel, album) with different classes and inheritance. 

Very important the library should contain a method listBooks which should display the all the books properties.

Your Tasks:

Please write a Java Application that contains classes that represent the models and behaviors explained above.

Homework must be written in Text Editor and not in IDEs (e.g Eclipse).

Finished homework must be uploaded to Moodle as a ZIP. (not RAR or anything else).
